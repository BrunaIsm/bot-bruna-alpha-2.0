# GitHub Pages Deploy

## Configuração do Frontend

1. Instalar gh-pages:
```bash
npm install --save-dev gh-pages
```

2. Atualizar package.json com homepage e scripts de deploy

3. Deploy:
```bash
npm run deploy
```

Frontend ficará em: https://brunaism.github.io/bot-bruna-alpha/

## Backend Local (por enquanto)

Continue usando o start-all.bat local.
Quando quiser hospedar o backend, use Railway.app (muito mais simples que Vercel).
